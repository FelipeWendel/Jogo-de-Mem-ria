Este jogo utiliza a biblioteca tkinter para criar uma interface gráfica simples. O jogo gera uma sequência de cores aleatórias e o usuário deve lembrar e repetir a sequência. Se o usuário acertar, o jogo gera uma nova sequência e aumenta a pontuação. Se o usuário errar, o jogo reinicia a sequência.

Como jogar

Execute o código acima.
Uma janela com botões de cores será exibida.
Observe a sequência de cores que será exibida na janela.
Tente repetir a sequência de cores clicando nos botões correspondentes.
Se você acertar, a pontuação será aumentada e uma nova sequência será gerada.
Se você errar, a sequência será reiniciada.

Observações

Este é um exemplo simples de um jogo de memória. Você pode personalizar o jogo adicionando mais cores, aumentando a dificuldade, etc.
A sequência de cores é gerada aleatoriamente, então cada vez que você jogar, a sequência será diferente.
